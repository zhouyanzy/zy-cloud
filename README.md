# zy-微服务后端项目

## 项目目录结构说明

```
├─zy
│  ├─basic--------------------------------------微服务基础模块
│  │  │─basic-auth------------------------------授权中心
│  │  │─basic-config----------------------------配置中心config
│  │  │─basic-eureka----------------------------注册中心eureka
│  │  └─basic-zuul------------------------------网关zuul
│  ├─blog---------------------------------------博客业务模块
│  ├─common-------------------------------------共通业务模块
│  ├─shop---------------------------------------商场业务模块
│  ├─util---------------------------------------工具模块
```
## 系统端口
 - `8760` basic-eureka
 - `8770` basic-zuul
 - `8780` basic-config
